# Lebanese Arabic Books Repository

## Directory Overview

This directory (`C:\Development\lebanese-arabic\books`) is a **Non-Code Project**. It serves as the master asset repository for the publishing arm of the "Lebanese Arabic" language learning business. 

The directory stores all source manuscripts, formatting templates, final export files (PDF, Mobi, HTML), cover art, and market research data for a series of educational language books. The overarching strategic goal of these assets is to transition the business toward a "Scalable Product" ecosystem, generating passive revenue and capturing high-intent, top-of-funnel traffic for the core tutoring service.

## Project Structure & Key Directories

The repository is organized into distinct project phases and volumes:

### 1. `Phrasebook Vol 1/`
This folder contains the assets for the first published book in the series.
*   **Purpose:** Houses the original drafts, marketing data (Amazon Ads from 2021, competitor analysis), and final publishing files.
*   **Technical Context:** This volume was originally drafted in MS Word. Due to older e-reader limitations with right-to-left (RTL) text, the Arabic script was embedded as images rather than actual text. 
*   **Key Subdirectories:**
    *   `book files/`: Contains crosswords, raw sentences in Excel, and original Word/CreateSpace templates.
    *   `Final ebook Documents/` & `Final Book Documents/`: The exported `.mobi`, `.html`, and `.pdf` files used for KDP publication.
    *   `market research/` & `Amazon Ads/`: Historical data on keywords, competition, and ad performance.

### 2. `Phrasebook Vol 2/`
This folder contains the assets for the second book, which focuses on advanced conversation and grammar.
*   **Purpose:** Stores the preparatory texts and professional layout files.
*   **Technical Context:** Unlike Vol 1, this book was designed using Adobe InDesign (`.indd`, `.indb`), allowing for vastly superior formatting, proper tabular layouts, and actual RTL Arabic text instead of images.
*   **Key Subdirectories:**
    *   `Preperations/`: Source files (`.docx`, `.xlsx`, `.txt`) containing the narrative-driven content (e.g., conversational texts, grammar rules).
    *   `Book files/`: The Adobe InDesign source files (`.indd`) broken down by chapter, and the compiled `.pdf` exports.

### 3. `story-books/`
A new, upcoming initiative to expand the product line.
*   **Purpose:** Research and planning for children's storybooks adapted for Lebanese Arabic learners.
*   **Key Files:** `non-poblematic-pd-story-list.txt` outlines a curated list of safe, public-domain stories (Aesop fables, global folklore) to be translated and adapted.

### 4. Root Level Tracking
*   `PHRASEBOOKS_HISTORY.md`: A living document tracking the project's history, current status, active sprints, and growth opportunities (e.g., migrating Vol 1 to InDesign, bundling strategies).

## Usage & Current Objectives

This repository is used by the publisher to manage the end-to-end lifecycle of the book portfolio—from initial market research and drafting in Word/Excel, to professional layout in InDesign, to final distribution formatting.

**Current Primary Objective:** 
The immediate active project within this repository is the **Volume 1 Design Migration**. The goal is to upgrade the legacy MS Word format of Vol 1 to match the professional Adobe InDesign standard of Vol 2. This involves replacing image-based Arabic script with actual text and redesigning tables. Once standardized, both volumes will be migrated off Amazon KDP to be sold directly via Bookvault and WooCommerce.
